-------------------------------------------------------------------------------------------------------------

------------------------------------------------------
| Pattern                 | Where it lives           |
| ----------------------- | ------------------------ |
| Factory                 | services / plugins       |
| Strategy                | notifications, workflows |
| Observer                | events                   |
| State                   | workflows / task_state   |
| Composite               | task hierarchy           |
| Decorator               | utils / plugins          |
| Command                 | services / events        |
| Chain of Responsibility | permissions              |
| Builder                 | search / analytics       |
| Facade                  | services                 |
------------------------------------------------------
